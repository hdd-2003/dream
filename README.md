# 梦境星河 🌌

> 记录一段梦境，点亮一颗星辰

一款精致的梦境记录与解读应用。把你的梦变成星辰，在星系中回溯每一次梦境的痕迹。

---

## ✨ 核心功能

### 📝 记录梦境
输入你的梦境内容，选择梦境基调（美梦 / 噩梦 / 奇幻 / 回忆），让AI帮你解读。

### 🔮 AI 解读
AI 会用诗意化的语言，为你解析梦境背后的含义，带来温暖而深刻的自我觉察体验。

### 🌟 星河可视化
每段梦境都会化作一颗独特的星辰，坠入你的专属星系。拖拽浏览、缩放探索，点击星辰重温旧梦。

---

## 🎨 梦境基调

| 类型 | 颜色 | 含义 |
|------|------|------|
| 美梦 ⭐ | 琥珀色 | 温馨、治愈 |
| 噩梦 🌙 | 紫色 | 恐惧、挑战 |
| 奇幻 ✨ | 青蓝渐变 | 自由、创造 |
| 回忆 ☁️ | 粉色 | 思念、眷恋 |

---

## 🎵 沉浸式体验

- 动态粒子星空背景
- 鼠标触碰粒子会散开
- 可开关的背景音乐
- 星辰呼吸式闪烁动画
